Features of OS

Among the features of the operating system are Protected and supervisor mode,
Allows disk access and file systems Device drivers Networking Security,
Program Execution, Memory management Virtual Memory Multitasking,
Handling I/O operations, Manipulation of the file system, Error Detection and
handling, Resource allocation, and Information and Resource Protection

Advantages of using Operating System Disadvantages of using Operating System
Allows you to hide details of hardware by creating an abstraction If any issue occurs in OS, you may lose all the contents which have been stored in your system
Offers an environment in which a user may execute programs/applications It is never entirely secure as a threat can occur at any time
It provides the computer system resources with easy to use format Operating system's software is quite expensive for small size organization which adds burden on them. Example Windows
